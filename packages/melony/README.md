# Melony

A modern React component library for building beautiful user interfaces.

## Installation

```bash
npm install melony
```

## Usage

```tsx
import { Button, Card, Input } from 'melony';

function App() {
  return (
    <Card>
      <Input placeholder="Enter your name" />
      <Button>Submit</Button>
    </Card>
  );
}
```
